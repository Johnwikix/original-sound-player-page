快速上手

从零开始：系统要求、安装方式、添加音乐库。

文本

系统要求

需要 Windows 10 1809 或更高版本（包括 Windows 11）。建议使用 Windows 11 以获得完整的 Mica / Acrylic 视觉效果。

文本

安装方式

从 Microsoft Store 下载安装可获得自动更新；如商店不可用，可通过 rg-adguard.net 获取离线 .msix 安装包（详见 FAQ）。

文本

添加音乐文件夹

应用启动后可通过四种方式添加音乐：点击「选择文件夹」按钮、拖拽文件夹到主窗口、在设置中启用「文件夹监听」自动检测、或使用「重新扫描」刷新已有库。

文本

开始播放

在歌曲列表中点击任意曲目即可开始播放。可使用底部播放栏的播放/暂停、上一首、下一首按钮控制。

音频设置

输出设备、缓冲区、DSD 处理、淡入淡出等音频相关配置。

下拉选择

输出设备

选择音频输出设备。下拉列表会枚举系统中所有可用的输出端（如扬声器、耳机、USB DAC）。

可选值:
WASAPI 独占(推送)WASAPI 独占(事件)WASAPI 共享WaveOutDirectSoundASIO
WASAPI 独占(推送)
数值输入100 ~ 900 (step 10) ms

缓冲区大小

音频缓冲区大小（单位：毫秒），范围 100-900。数值越小延迟越低，但可能引起爆音；数值越大越稳定。

500ms
下拉选择

DSD 转 PCM 频率

将 DSD 音频流转换为 PCM 时使用的采样率。仅当 DSD 模式为「DoP」时生效。

可选值:
44.1 kHz48 kHz88.2 kHz96 kHz176.4 kHz192 kHz352.8 kHz384 kHz
44.1 kHz
数值输入0 ~ 50 (step 1) dB

DSD 增益

DSD 信号到 PCM 的增益补偿（单位：dB），可降低 DSD→PCM 转换后的音量损失。仅 DoP 模式生效。

25dB
开关

DSD DoP / DSD Native

切换 DSD 输出模式。DoP（DSD over PCM）将 DSD 数据打包进 PCM 帧传输，兼容多数 DAC；Native 直接输出原始 DSD 流，需要 DAC 硬件支持。

关 / 开
开关

淡入淡出

切换歌曲时启用音量淡入淡出效果，使衔接更平滑自然。

关 / 开
