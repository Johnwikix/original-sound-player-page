## 原音HQ播放器页面重新设计方案

### 1. 安装动画库依赖
- 安装 GSAP (GreenSock Animation Platform) + ScrollTrigger
- 用于实现高性能滚动动画

### 2. Hero 区域重新设计
- **移除轮播图**，改为单一动态背景
- 添加**粒子/星空动态背景**效果（使用CSS或canvas）
- 标题文字添加**交错渐入动画**（每个字依次出现）
- 副标题使用**淡入上移**效果
- 按钮组添加**悬停发光/光晕效果**
- 添加**视差浮动的装饰元素**（如音符、波浪线条）

### 3. 视频展示区优化
- 与Hero区域视觉上融合（无明显分隔）
- 移除边框，增强**投影和发光效果**
- 添加**悬停放大**动画

### 4. 功能卡片区重新设计
- 使用 **CSS 3D transform** 实现卡片悬停时的**微倾斜效果**
- 添加**滚动触发入场动画**（stagger效果，卡片依次出现）
- 卡片背景添加**渐变光晕**
- 图片添加**缩放悬停效果**

### 5. 下载/可视化区重新设计
- 添加**视差滚动的背景元素**
- 频谱可视化区域添加**动态发光效果**
- 添加**浮动装饰元素**

### 6. 页面整体动画
- 添加**平滑滚动**
- 各区块添加**滚动触发的渐入动画**
- 页面加载时的**开场动画**

### 7. 视觉风格升级
- 增强渐变色彩（紫色/蓝色/青色主题）
- 添加**玻璃态效果**（glassmorphism）
- 添加更多**发光和光晕效果**
- 优化响应式设计

### 8. 关键技术实现
- 使用 GSAP ScrollTrigger 实现滚动动画
- 使用 CSS 3D transforms 实现倾斜效果
- 使用 CSS animations 实现粒子背景
- 保持原有 i18n 多语言支持
